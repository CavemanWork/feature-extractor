feature-extractor
=================

Feature Extraction program with Grammatical Evolution.
We have FE_Base, and gogenpy package
gogenpy package contains of several modules.
* utils.py contains:
  - object:
	 - randomizer
  - functions:
	 - bin_to_dec
	 - dec_to_bin
	 - bin_digit_needed
	 - execute
	 - plus
	 - minus
	 - multiply
	 - divide
* classes.py contains:
  - classes:
  	 - Tree
  	 - GA_Base
  	 - Genetics_Algorithm
  	 - Grammatical_Evolution
  	 - Genetics_Programming
